# Task Manager - Demo Script & Checklist

Use this checklist when recording your demo video.

## Pre-Demo Setup

- [ ] Backend running on localhost:5000
- [ ] Frontend running on localhost:5173
- [ ] MongoDB connected
- [ ] Browser console closed (F12)
- [ ] Browser zoomed to 100%
- [ ] Screen recorded in 1080p or higher

## Demo Flow (2-5 minutes)

### 1. Authentication (30 seconds)
- [ ] Show signup page
- [ ] Create new account with:
  - Name: Demo User
  - Email: demo@taskmanager.com
  - Password: demo123456
- [ ] Show successful signup
- [ ] Logout
- [ ] Show login page
- [ ] Login with credentials
- [ ] Show dashboard after login

### 2. Project Management (1 minute)
- [ ] Show projects list (empty or existing)
- [ ] Click "+ New Project"
- [ ] Create project named "Marketing Campaign"
- [ ] Add description: "Q1 marketing campaign planning and execution"
- [ ] Submit and show project created
- [ ] Click on project to view details

### 3. Task Management (1.5 minutes)
- [ ] Show empty tasks list
- [ ] Click "+ New Task"
- [ ] Create first task:
  - Title: "Create social media strategy"
  - Description: "Plan posts for all platforms"
  - Priority: High
  - Due Date: 2 weeks from now
- [ ] Create second task:
  - Title: "Design brand assets"
  - Priority: High
  - Due Date: 1 week from now
- [ ] Create third task:
  - Title: "Set up email campaign"
  - Priority: Medium
- [ ] Show all tasks in list

### 4. Task Updates (1 minute)
- [ ] Click on a task status dropdown
- [ ] Change "Create social media strategy" to "In Progress"
- [ ] Change "Design brand assets" to "Done"
- [ ] Show dashboard updating in real-time
- [ ] Point out:
  - Total tasks: 3
  - To Do: 1
  - In Progress: 1
  - Done: 1

### 5. Dashboard & Statistics (30 seconds)
- [ ] Scroll to dashboard
- [ ] Show statistics:
  - Total Tasks
  - Tasks by Status breakdown
  - Overdue Tasks count
- [ ] Explain each metric

### 6. Role-Based Access (Optional, 30 seconds)
- [ ] Explain Admin role (creator)
- [ ] Explain Member role (in production)
- [ ] Show that only admin can delete projects/tasks

## Talking Points

Include in your explanation:
1. **Problem Solved**
   - "Team collaboration tool for task management"
   - "Like Trello but simpler and custom-built"

2. **Key Features**
   - Real-time task updates
   - Role-based access control
   - Comprehensive dashboard
   - Secure authentication

3. **Tech Stack**
   - Frontend: React + Vite
   - Backend: Node.js + Express
   - Database: MongoDB
   - Deployment: Railway

4. **Deployment**
   - Both frontend and backend deployed on Railway
   - Live URL: [Your deployed URL]
   - Available publicly

5. **Future Enhancements** (if asked)
   - Task comments/discussion
   - File attachments
   - Notifications
   - Mobile app
   - Team collaborations

## Video Quality Checklist

- [ ] Audio is clear and not too loud
- [ ] No background noise
- [ ] Screen is readable (text not too small)
- [ ] Video is steady (use tripod or stable mount)
- [ ] Lighting is adequate
- [ ] Transitions are smooth
- [ ] Total length: 2-5 minutes
- [ ] File format: MP4 or WebM
- [ ] Resolution: 1080p (1920x1080) minimum

## Screen Recording Tips

### On Windows:
- Use Windows Game Bar (Win+G)
- Or use OBS Studio (free)
- Or use ScreenFlow

### On macOS:
- Use QuickTime Player (Cmd+Shift+5)
- Or use ScreenFlow
- Or use OBS Studio

### On Linux:
- Use OBS Studio
- Or use SimpleScreenRecorder
